Virtual assistant in python

In my 10th-grade year, I developed a Virtual Assistant for PC using Python, integrating Google's Natural Language Processing (NLP) library for audio input. Even though it was a simple python script, the program boasted several standout features, setting it apart from similar projects found on YouTube. Users could perform tasks like sending emails, attaching files, playing music, and launching applications, making it a highly versatile and productive tool.

Key Features:

Audio Input and NLP: The Virtual Assistant supported audio commands through Google's NLP, ensuring a natural and intuitive user experience. Email Functionality: Users could send emails with ease, as the Virtual Assistant was capable of drafting and sending messages. Attachment Handling: The assistant seamlessly managed attachments, allowing users to share files and documents with their contacts. Media Playback: Users enjoyed the convenience of requesting the Virtual Assistant to play music and other media files. Application Launching: The Virtual Assistant responded to voice commands to open any application on the user's PC.

Achievements:

The project's success caught the attention of Witblox Techfest, in collaboration with IIT Bombay, where I secured a spot among the top 15 projects nationwide. My Virtual Assistant stood out due to its advanced features and practical usability, setting it apart from others on YouTube.

Conclusion:

Creating the Virtual Assistant for PC was an exciting journey, allowing me to explore NLP, Python, and integrating different libraries in one single project. The project's recognition at Witblox Techfest affirmed my passion for programming and AI.
